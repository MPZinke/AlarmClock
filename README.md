# AlarmClock
Python Alarm Clock to run on Raspberry Pico with eInk display.


## Features

### Key Features

- Clock
- Alarm
- ePaper Display


### Future Features
- Back Light
- Multiple Alarms
- Multiple Alarm Songs
- White Noise


## Obstacles
- Delay of screen (~2 seconds)
- Memory constraints
	- Screen displaying
	- Sound queuing
- Hardware 
	- Compatibility (having enough pins & sharing pins)
	- Packaging


## Libraries
- DFRobotDFPlayerMini


## Board
- Arduino Mbed OS RP2040 Boards > Raspberry Pi Pico
